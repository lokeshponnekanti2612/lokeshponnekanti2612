# 🧠 About Me

I am an AI/ML Engineer specializing in **production-grade Generative AI systems**, Retrieval-Augmented Generation (RAG), embeddings infrastructure, and scalable LLM inference.

My focus is not just building AI demos — but engineering **reliable, modular, observable AI systems** that operate in real-world environments with latency, cost, and scale constraints.

I design systems end-to-end:
> Data Ingestion → Chunking → Embeddings → Vector Index → Retrieval → LLM Inference → API Deployment
